# @milkdown/plugin-trailing

Add a trailing node at the end of the document automatically.

```typescript
import { Editor } from '@milkdown/core';

import { trailing } from '@milkdown/plugin-trailing';

Editor
  .make()
  .use(trailing)
  .create();
```
