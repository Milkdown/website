# @milkdown/plugin-indent

Indent support for [milkdown](https://milkdown.dev/).

```typescript
import { Editor } from '@milkdown/core';
import { indent } from '@milkdown/plugin-indent';

Editor
  .make()
  .use(indent)
  .create();
```
